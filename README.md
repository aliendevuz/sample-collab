# Sample sandbox environment

Bu loyihaning asosiy maqsadi jamoada tasklarga asoslanib topshiriqni bajarish orqali loyihani ishlab chiqarishni o'rganish

Buni amalga oshirish uchun har kim o'ziga oddiy html based shaxsiy veb sahifa yaratishi kerak bo'ladi, 2 kun davomida etadi. home sahifasida barcha sahifalar uchun havolalar bo'ladi, havolalar bosilganda har bir ishtirokchi yaratgan shaxsiy sahifa ochilishi kerak, 1 - kun sahifani ochiladigan holatga keltiramiz va bir birimizni qilgan ishlarimizni tekshirib feedbacklarni issue orqali beramiz, 2 - kun esa feedback asosida tuzatishlarni amalga oshiramiz, va kimda muammo bo'lsa birgalikda ko'rib chiqamiz

git clone https://github.com/aliendevuz/sample-collab.git sample
